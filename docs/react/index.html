React结合

React iFlow

它是一个React和iFlow的状态连接库

React iFlow 如果你使用React和iFlow，那么你将发现iFlow可以让你默认情况几乎不用写选择器，同时如果有必要，iFlow连接器也支持多种选择器写法。

🔥🔥🔥这是一个高效和简洁的React和iFlow连接器🔥🔥🔥

特性和优点

  • 尽可能少的选择器，甚至不用选择器
  • 自动 Array Diff
  • 支持各种选择器写法
