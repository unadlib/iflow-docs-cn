getStore

getStore 方法

描述

get用于快速在Pipe通过path得到当前节点下的对应path的值

用法

getStore(store, [path])

参数

store(Object/Array): 父级store path(String/Array): 需要取值的path

返回值

(*): 返回需要取值的path的值

示例

import iFlow,{ getStore } from 'iflow'
const pipe = iFlow({
  calculate: external(async function (number) {
    // do async something
  }),
  counter: 0,
  foo: {
    bar: 88
  },
})
const store = pipe.create()
getStore(store, ['foo', 'bar']) // value: 88
